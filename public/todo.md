## שלב 5 - שדרוגים מתקדמים, אוטומציה וחוויית משתמש (Level PRO)
- [ ] **אינטגרציית זמנים חכמה (Zmanim API)**: שילוב API לזמני כניסת חג, סוף זמן אכילת חמץ, חצות הלילה (לאפיקומן) וספירת העומר לפי מיקום.
- [ ] **מערכת התראות (Push Notifications)**: התראות בזמן אמת לפני זמנים חשובים, תזכורות למשימות וספירת העומר.
- [ ] **אלגוריתם איזון עומסים במשימות**: חלוקה חכמה של משימות הניקיון על פני הימים שנותרו כדי למנוע עומס (Load Balancing).
- [ ] **אופטימיזציית קניות חכמה**: איחוד מצרכים כפולים ברשימת הקניות ואלגוריתם לצמצום בזבוז מזון.
- [ ] **קבוצות משפחתיות פרטיות (Family Hub)**: אזור סגור למשפחה לשיתוף משימות, רשימות קניות משותפות ותכנון הסדר יחד.
- [ ] **מערכת ניהול אורחים**: שליחת הזמנות דיגיטליות לליל הסדר, אישור הגעה (RSVP) ואיסוף רגישויות למזון.
- [ ] **עיצוב חווייתי "מסע יציאת מצרים"**: שדרוג ה-UX עם אנימציות מעבר, ערכות נושא (זהב/לבן/כחול) ותחושת התקדמות ויזואלית.
- [ ] **שקלול ציון רוחני מתקדם**: אלגוריתם המשלב את נתוני הניקיון (30%), הלימוד (40%) וההשתתפות בסדר (30%) לציון התקדמות כולל.
- [ ] **משחק "מצא את האפיקומן" מתקדם**: פיתוח משחק אינטראקטיבי עשיר יותר לילדים (אולי בשילוב מציאות רבודה AR).
- [ ] **סנכרון וגיבוי מתקדם**: סנכרון מלא בזמן אמת בין מכשירים (למשל, שני בני זוג מעדכנים את רשימת הקניות במקביל).
