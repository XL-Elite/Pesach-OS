# Pesach OS ✨

מערכת חכמה, אינטראקטיבית וניהולית לחג הפסח.
Pesach OS נועדה להפוך את ההכנות לחג, את ליל הסדר ואת ימי החג לחוויה מאורגנת, רוחנית ומהנה לכל המשפחה.

## 🔗 קישור לאפליקציה (Live Demo)
[לחץ כאן כדי להיכנס למערכת](https://ai.studio/apps/ad4135cd-2100-4ff3-913f-3f4a0b07c631?fullscreenApplet=true)

## 🚀 תכונות עיקריות
- **ניהול ליל הסדר אינטראקטיבי**: ליווי מלא של 15 סימני הסדר עם רשימות משימות, ווארטים, טיימר מובנה ומצב ילדים מיוחד.
- **ספריית ידע מקיפה**: מאגר עצום של הלכות, מדריכי ניקיון, מתכונים, פיוטים ופירושים להגדה.
- **שולחן דיונים (Group Chat)**: צ'אט קבוצתי בזמן אמת לכל משתמשי המערכת.
- **ניהול משימות (Todo)**: רשימות מטלות חכמות לניקיונות, קניות והכנות לחג.
- **ממשק משתמש מתקדם**: עיצוב מודרני, רספונסיבי ותומך בעברית (RTL).
- **ניהול משתמשים**: התחברות מאובטחת באמצעות Google, פרופילים אישיים ופאנל ניהול למנהל המערכת.

## 🛠️ טכנולוגיות
- **Frontend**: React, Vite, Tailwind CSS.
- **Backend/Database**: Firebase (Authentication, Firestore).
- **AI**: שילוב מתקדם של Gemini API.
