# רשימת קניות אולטימטיבית לפסח

## 🍷 קערת הסדר ומצוות הלילה
- [ ] מצות שמורות עבודת יד (לפחות 3 לכל קערת סדר)
- [ ] יין אדום (לפחות 4 כוסות לכל מבוגר)
- [ ] מיץ ענבים (לילדים ולמי שמתקשה ביין)
- [ ] חסה / עלי אנדיב (למרור)
- [ ] שורש חזרת (למרור)
- [ ] כרפס / תפוח אדמה / צנון (לכרפס)
- [ ] זרוע (כנף עוף או גרון צלוי)
- [ ] ביצים (לקערה ולסעודה - הרבה!)
- [ ] תפוחים, אגוזים, שקדים, תמרים, יין מתוק (להכנת חרוסת)
- [ ] מלח (למי מלח)

## 🍞 מוצרי יסוד ויבשים (כשל"פ)
- [ ] מצות רגילות (מכונה)
- [ ] קמח מצה (לאוכלי שרויה)
- [ ] קמח תפוחי אדמה
- [ ] קמח שקדים / שקדים טחונים
- [ ] שמן (אגוזים, דקלים, או קנולה/סויה לאוכלי קטניות)
- [ ] סוכר, מלח, פלפל שחור, תבלינים כשל"פ
- [ ] קפה, תה, קקאו
- [ ] ממרח שוקולד / ריבה
- [ ] חטיפים לפסח (במבה כשל"פ, צ'יפס, עוגיות בוטנים/קוקוס)

## 🥩 בשר, עוף ודגים
- [ ] עופות שלמים / כרעיים / חזה עוף
- [ ] בשר בקר לבישול ארוך (צלי, אסאדו)
- [ ] בשר טחון
- [ ] דגים (אמנון, סלמון, נסיכת הנילוס, קרפיון לגפילטע פיש)

## 🥦 ירקות ופירות (טריים)
- [ ] תפוחי אדמה (המון!)
- [ ] בצל, שום
- [ ] גזר, קישואים, בטטות, דלעת (למרק ולתוספות)
- [ ] עגבניות, מלפפונים, פלפלים
- [ ] ירק: פטרוזיליה, כוסברה, שמיר, סלרי
- [ ] פירות העונה (תפוחים, בננות, תפוזים, אגסים)

## 🥛 מוצרי חלב וביצים (כשל"פ)
- [ ] חלב
- [ ] גבינות לבנות / קוטג'
- [ ] גבינה צהובה
- [ ] חמאה
- [ ] יוגורטים / מעדנים לילדים
- [ ] תבניות ביצים (לפחות 2-3 תבניות למשפחה ממוצעת)

## 🧽 חומרי ניקוי וחד-פעמי
- [ ] אקונומיקה, נוזל רצפות, מסיר שומנים
- [ ] סבון כלים כשל"פ
- [ ] ספוגים לכלים (חדשים לפסח)
- [ ] נייר כסף (רגיל ועבה לציפוי שיש)
- [ ] ניילון נצמד, נייר אפייה
- [ ] צלחות, כוסות, סכו"ם חד-פעמי (מהודר לסדר, רגיל לחול המועד)
- [ ] מפות שולחן (חד פעמיות או בד)
- [ ] מפיות חגיגיות

## 🎁 שונות
- [ ] אגוזים בקליפתם (למשחקי ילדים)
- [ ] מתנות לאפיקומן
- [ ] מתנות למארחים / לבני הזוג
- [ ] נרות שבת וחג, נר נשמה (ליזכור ולשביעי של פסח)
- [ ] הגדות של פסח (לוודא שיש מספיק לכל האורחים)
